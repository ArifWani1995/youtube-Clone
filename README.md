

🔹 Core Requirements:
1. Design a clean, modern, and visually appealing UI inspired by YouTube.
2. Fully responsive (mobile, tablet, desktop).
3. Use smooth animations and transitions.

🔹 Frontend Features:
- Home page with video feed (grid layout)
- Sidebar navigation (Home, Trending, Subscriptions, Library)
- Top navbar with:
  - Logo
  - Search bar (real-time filtering)
  - User profile icon
- Video cards showing:
  - Thumbnail
  - Title
  - Channel name
  - Views and upload time

🔹 Video Player Page:
- Embedded video player (HTML5 or iframe)
- Title, views, likes/dislikes buttons
- Channel info with subscribe button
- Comment section (add, delete comments)
- Suggested videos sidebar

🔹 Advanced Features:
- Dark mode toggle
- Like/Dislike system (with counters)
- Subscribe/Unsubscribe functionality
- Video upload feature (with preview)
- Search functionality with filtering
- Category-based browsing (Music, Gaming, Education, etc.)

🔹 Backend (Optional but Recommended):
- User authentication (login/signup)
- Store videos, users, comments using:
  - Firebase / MongoDB
- REST API for:
  - Upload video
  - Fetch videos
  - Add comments
  - Like/subscribe actions

🔹 Technical Stack:
- HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)
- Bootstrap or Tailwind CSS (for styling)
- Optional: React.js for frontend
- Optional: Node.js + Express for backend
- Optional: Firebase (Auth + Firestore + Storage)

🔹 UI/UX Enhancements:
- Hover effects on video thumbnails
- Loading skeletons
- Infinite scrolling
- Smooth transitions and animations
- Attractive color palette (light + dark themes)

🔹 Extra Features (for uniqueness):
- AI-based video recommendations (basic logic)
- Watch history tracking
- Playlist creation
- Video categories filter
- Notification bell icon UI

🔹 Deliverables:
- Well-structured code (separate files)
- Clean folder structure
- Commented code for understanding
- Fully working demo (frontend + backend if included)

🔹 Goal:
Build a professional-level project suitable for portfolio and real-world usage, with a polished UI and interactive user experience.
